ImageZoomer is a Windows Forms application that allows users to zoom in and zoom out of an image with simple button controls.  
The program displays an image inside a PictureBox and provides two buttons for enlarging and shrinking the picture smoothly.

Features
- Zoom In: Increase the image size  
- Zoom Out: Decrease the image size  
- Simple and clean Windows Forms interface  

ImageZoomer is a minimal, fast, and easy-to-use tool for viewing images with adjustable size.
